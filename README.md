# social_pulse - trend analyzer dashboard

## Project preview
- I've been working for this project for a while. A Web apps that fetches **Real Google Trends Data**, analyzes it, and displays it on a beautiful Dashboard. Social Pulse — Trend Analyzer Dashboard (Python, Flask, SQLite, HTML/CSS). Built a real-time web dashboard that fetches and visualizes Google Trends data by category and country. Implemented a SQLite database to store trend history and used java script for interactive visualizations.

## Overview of the Project 
- Uses a real time API ( not a dead CSV )
- Has a working web interface.
- Shows Python + Web + Data + SQL all in one.

## What it Does 
- Fetches trending search topics by category & country using Google Trends.
- Shows which topics are Rising vs Falling.
- Compares two trends overtime.
- Stores search history in a SQLite database.
- Displays everything on a clean HTML/CSS dashboard.

## Tech Stack
- pytrends - Data fetching (Google Trends Python library)
- Backend - Flask (Python web framework)
- Database - SQLite (store trend history)
- Frontend - HTML + CSS + js
